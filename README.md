# CodeX Editor
Next generation Editor for modern web applications.

### Block-styled

Entry composed by different and extended list of Blocks

### JSON as output

Outputs clear Blocks data via JSON

```json
{
  "blocks": [
    {
      "type": "paragraph",
      "data": {
        "text": "There is a first paragraph"
      }
    },
    {
      "type": "header",
      "data": {
        "text": "Aaand here is a Header",
        "type": "H2"
      }
    },
   ...
  ]
}
```
